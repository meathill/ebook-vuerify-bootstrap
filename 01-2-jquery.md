# 石器时代 - jQuery

这个时候浏览器还处于非常严重的分裂状态。IE 自恃有 Windows 护身，真是想怎么搞怎么搞。另一边的网景后来的 Firefox 空有一腔报国热血，但是怎么都干不过 IE。后来干脆全捐了，曲线救国，用免费开源的群众战争和微软打。

这种环境受苦的还是网页开发者。同一个页面，经常要写两套代码，然后寻找各式各样的 Hack 方法，费时费力。于是 jQuery 出现后大家觉得简直是一股清流。
